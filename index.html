LoRA · Fine-Tuning

PromptJob

PromptJob is an AI assistant built by fine-tuning Google's Gemma 3 1B model with LoRA. Instead of answering questions directly, it takes a user's rough prompt and rewrites it into a clearer, more effective version — helping people get noticeably better responses out of other LLMs. The project covers the full fine-tuning pipeline: dataset preparation, LoRA training, and packaging the result with Hugging Face.

RAG · LangChain · Docker

Intelligent RAG Knowledge Assistant

An end-to-end Retrieval-Augmented Generation system built with LangChain and Ollama. Documents are chunked and embedded for semantic search, then relevant context is retrieved and passed to the LLM so answers stay grounded in the source material rather than hallucinated. The assistant is exposed through a FastAPI REST API and packaged with Docker for reproducible, containerized deployment.

CrewAI · Agentic AI

Autonomous Multi-Agent AI System

A collaborative multi-agent workflow built with CrewAI and LangChain, where separate agents take on distinct roles — planning, research, execution, and validation — and coordinate to complete complex tasks with minimal human input. The system demonstrates how breaking a task into specialised agents can produce more reliable outcomes than a single monolithic prompt.

Neural TTS

Zpeak

Zpeak is a neural text-to-speech web app that turns written text into natural-sounding voice. It combines Edge TTS, Kokoro, and Supertonic models so you can pick between fast, lightweight synthesis or higher-fidelity voices depending on the use case. The frontend runs on GitHub Pages while the actual inference is served through a Hugging Face Space, keeping the app free to host and easy to update.

Text-to-Image

Diego Imagine

Diego Imagine generates images directly from a text prompt using SD-Turbo, a distilled Stable Diffusion model built for near-instant generation. The goal was to make image generation feel responsive rather than a multi-second wait, so prompts are rendered in just a few steps. Like the other apps, the UI is hosted on GitHub Pages and calls a Hugging Face Space for the actual model inference.

Text-to-Music

Hans Music

Hans Music turns a short text description into an original piece of music using Meta's MusicGen model. Describe a mood, genre, or instrumentation and the app generates an audio clip that matches the prompt — useful for quick background scoring or just experimenting with generative audio. As with the other tools, the frontend is static and the heavy generation work happens on a Hugging Face Space.
